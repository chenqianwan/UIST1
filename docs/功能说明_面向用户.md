# 合同星座 · 功能说明（面向用户）

本文档描述「合同星座」产品的主要功能，方便用户了解能做什么、怎么用。

---

## 一、主界面概览

产品分为两个入口：

- **主工作台**：默认进入，用于搭建和编辑合同条款图谱。
- **管理员看板**：通过地址栏 `#/admin` 进入，用于查看使用热力图等统计（面向管理员）。

以下说明均针对**主工作台**。

---

## 二、节点库（左侧 / 右侧面板上方）

- **作用**：提供可用的合同条款模板，作为搭建合同的「积木」。
- **内容**：每个模板展示名称、类型图标、风险等级（无风险 / 低 / 中 / 高）、简短说明。
- **操作**：
  - **拖拽**：将某个模板从节点库拖到中间画布，松开后会在画布上生成对应节点（或一整条演示链）。
  - 已被使用过的模板会从库中移除，避免重复添加。

---

## 三、画布（中间区域）

- **作用**：展示整份合同的结构图，节点表示条款，连线表示条款之间的从属或引用关系。
- **节点**：
  - 不同**风险等级**用不同颜色区分（如绿 / 黄 / 橙 / 红）。
  - 中心有一个「主合同」根节点，其余节点通过连线与根或彼此连接。
- **操作**：
  - **点击画布空白处**：取消当前选中。
  - **点击某个节点**：选中该条款，右侧「当前选中节点」区域会显示其详情与修改建议。
  - **拖拽节点**：可移动节点位置；若拖到底部**回收区**并松开，会删除该节点及其下属子节点。

---

## 四、当前选中节点（右侧面板）

选中画布上的某个节点后，右侧会显示该节点的详细信息和可执行操作。

### 4.1 基本信息

- 条款名称、风险等级、正文内容。
- 若有已完成的修改，会提示「AI 更新已应用」等状态。

### 4.2 建议修改（仅对有风险的条款显示）

系统会为**有风险等级**的条款生成修改建议，分为三类：

| 类型       | 含义说明 |
|------------|----------|
| **删除**   | 该条款在现有表述下风险较高，建议整体删除。 |
| **修订**   | 建议调整措辞，使条件更清晰、歧义更少。 |
| **补充条款** | 建议增加补充说明，堵住潜在漏洞。 |

每条建议会展示：
- 建议类型与简短原因；
- 若为修订：展示建议替换的文案；
- 若为补充：展示建议补充的条款草稿。

### 4.3 单条执行

- 每条建议旁有对应按钮（如「删除该条款」「应用修订」「添加补充条款」）。
- 点击后**仅对这一条建议**生效：或删除节点，或替换内容，或新增子节点并合并内容。
- 执行后该条款风险可能变为「无风险」，对应建议会消失。

### 4.4 一键递归修改

- 当当前选中节点及其**下属子节点**中存在至少一条待执行的建议时，会显示「一键递归修改（共 X 项）」按钮。
- 点击后先弹出**确认框**，汇总将影响的节点数量以及各类操作数量（先删除、再修订、再补充）。
- 确认后：
  - 会先执行所有**删除**（被删节点及其子节点不再参与后续操作）；
  - 再对剩余节点执行**修订**与**补充**。
- 执行过程中按钮会显示「处理中…」，结束后会短暂提示「已递归执行 X 项」。

**规则说明**：若某节点存在「删除」建议，则对该节点只执行删除，不再执行修订或补充。

---

## 五、导出合同

- **位置**：右侧面板底部，「导出合同」按钮。
- **作用**：根据当前画布上的条款与结构，生成并导出合同内容（演示场景下会模拟导出成功）。
- **状态**：导出中会显示「Exporting...」，成功后会有简短成功提示。

---

## 六、回收区（画布底部）

- **位置**：画布下方「Drop / Click Here」区域。
- **作用**：删除不需要的节点。
- **操作**：
  - **拖拽**：将画布上的节点拖到该区域并松开，会删除该节点及其所有子节点。
  - **点击**：在已选中某节点的情况下，点击该区域也会删除当前选中节点及其子节点。

---

## 七、分析控制（画布左下）

- **位置**：画布左下角「Analysis Controls」面板。
- **内容**：三个滑块，用于调整画布上节点的排布倾向（不改变条款内容，只影响视觉布局）：
  - **Semantic Pull（语义拉力）**：按条款语义相似度分组排布。
  - **Risk Pull（风险拉力）**：按风险等级分列排布。
  - **Time Pull（时间拉力）**：按条款所处时间阶段（如签约前、执行期、终止后等）分列排布。
- 滑块从 0% 到 100%，数值越大，对应维度的排布效果越明显。底部有风险等级图例（无风险 / 低 / 中 / 高）便于对照。

---

## 八、管理员看板（#/admin）

- **入口**：在浏览器地址栏将路径改为 `#/admin` 后回车即可进入。
- **用途**：面向管理员，用于查看产品使用情况的可视化统计。
- **功能概览**：
  - 展示用户在画布、右侧修改区、导出区、回收区、分析控制区等区域的**操作热力图**；
  - 区分点击、拖拽、悬停等不同交互；
  - 可查看事件列表与时间线，便于分析使用习惯或排查问题。

---

## 九、功能一览表

| 功能           | 说明 |
|----------------|------|
| 从节点库拖拽添加条款 | 将模板拖到画布生成节点或演示链。 |
| 点击选中节点       | 右侧显示条款详情与修改建议。 |
| 单条执行建议       | 对某一条建议执行删除 / 修订 / 补充。 |
| 一键递归修改       | 对当前节点及子节点批量执行建议，先删再修订再补充，带确认与进度提示。 |
| 拖拽/点击回收区删除  | 删除节点及其子节点。 |
| 导出合同          | 按当前结构导出合同内容。 |
| 分析控制滑块       | 按语义、风险、时间维度调整节点排布。 |
| 管理员看板         | 热力图与事件统计（#/admin）。 |
